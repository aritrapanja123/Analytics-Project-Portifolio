# Optimisation and Decision Modelling

## Project Overview

This project applies Linear Programming to a student transportation and school allocation problem. The objective is to minimise transport cost while satisfying capacity, allocation, and policy constraints.

## Business Problem

Following a school closure, students must be redistributed across remaining schools. Decision-makers need to balance transport cost with school capacities and allocation requirements.

## Objectives

- Formulate the allocation problem mathematically.
- Define decision variables and constraints.
- Minimise total transportation cost.
- Ensure all students are allocated.
- Respect school capacity limitations.
- Evaluate alternative scenarios.

## Optimisation Model

The model uses decision variables representing the number of students assigned from residential areas to available schools.

The model includes:

### Objective Function

Minimise total transportation cost.

### Constraints

- all students must be allocated;
- school capacities cannot be exceeded;
- grade-distribution requirements must be satisfied;
- decision variables must remain non-negative.

## Solution Approach

The optimisation model is implemented using Excel Solver. Scenario analysis is used to examine how changes in assumptions or constraints affect the recommended allocation and total cost.

## Business Value

The project demonstrates how mathematical optimisation can support transparent and evidence-based resource allocation.

The same modelling principles can be transferred to:

- logistics network design;
- warehouse allocation;
- workforce scheduling;
- production planning;
- inventory allocation;
- transportation planning.

## Tools and Techniques

Excel Solver, Linear Programming, optimisation modelling, constraint formulation, scenario analysis, and quantitative decision support.

## Skills Demonstrated

Problem formulation, objective-function design, constraint modelling, resource allocation, scenario analysis, optimisation, and interpretation of quantitative outputs.

## Future Improvements

The model could be extended with integer programming, multi-objective optimisation, travel-time constraints, emissions objectives, uncertainty analysis, and interactive scenario dashboards.
