# Advanced Analytics: Customer Sentiment Analysis

## Project Overview

This project applies Natural Language Processing and machine learning to customer review data to classify sentiment and compare different analytical approaches.

## Business Problem

Customer reviews contain valuable information about satisfaction and service quality, but manual review becomes impractical at scale. Organisations need analytical methods that can automatically identify sentiment patterns and emerging customer experience issues.

## Objectives

- Prepare unstructured review text for analysis.
- Classify customer reviews by sentiment.
- Compare traditional machine learning and deep learning approaches.
- Evaluate model performance.
- Consider how sentiment analytics can support business decisions.

## Analytical Approach

The project applies a structured NLP workflow including:

- text cleaning;
- tokenisation;
- stop-word removal;
- text vectorisation;
- model training;
- model comparison;
- performance evaluation.

Models and techniques explored include:

- TF-IDF;
- Logistic Regression;
- Multinomial Naive Bayes;
- Convolutional Neural Networks;
- BERT.

## Business Value

Automated sentiment analysis can help organisations monitor customer satisfaction at scale and identify recurring issues more quickly.

Potential applications include:

- customer experience monitoring;
- complaint prioritisation;
- service improvement;
- product feedback analysis;
- trend detection across large review datasets.

## Tools and Techniques

Python, Natural Language Processing, TF-IDF, Logistic Regression, Naive Bayes, CNN, BERT, text preprocessing, classification, and model evaluation.

## Skills Demonstrated

Unstructured data analysis, NLP pipeline design, predictive modelling, deep learning awareness, transformer model application, comparative evaluation, and business interpretation of analytical results.

## Future Improvements

Future extensions could include aspect-based sentiment analysis, multilingual sentiment detection, topic modelling, real-time review monitoring, and deployment through a dashboard or API.
